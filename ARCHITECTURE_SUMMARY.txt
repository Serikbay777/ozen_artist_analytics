ТЕКУЩАЯ СТРУКТУРА (Упрощенная):

    Пользователь
         │
         ▼
    ┌─────────────────┐
    │  ORCHESTRATOR   │  ← Анализирует вопрос
    └────────┬────────┘
             │
             ▼
    ┌─────────────────┐
    │  VERIFICATION   │  ← Читает verification.txt
    │     AGENT       │     и отвечает
    └─────────────────┘
             │
             ▼
         Ответ

═══════════════════════════════════════════════════════════════
                        КОМПОНЕНТЫ
═══════════════════════════════════════════════════════════════

1. OrchestratorAgent (app/agents/OrchestratorAgent.py)
   → Проверяет что вопрос про верификацию
   → Направляет к нужному агенту

2. VerificationAgent (app/agents/faq/VerificationAgent.py)
   → База знаний: app/services/faq/verification.txt
   → Отвечает на вопросы о:
     • Apple Music for Artists
     • Spotify for Artists
     • BandLink x Яндекс Музыка
     • VK Studio

3. WorkflowManager (app/agents/WorkflowManager.py)
   → Управляет LangGraph workflow
   → Связывает агентов

═══════════════════════════════════════════════════════════════
                    КАК ДОБАВИТЬ АГЕНТА
═══════════════════════════════════════════════════════════════

Шаг 1: Создать файл базы знаний
   app/services/faq/new_topic.txt

Шаг 2: Создать агента
   app/agents/faq/NewTopicAgent.py
   (скопировать структуру из VerificationAgent.py)

Шаг 3: Обновить OrchestratorAgent.py
   Добавить описание нового агента в промпт

Шаг 4: Обновить WorkflowManager.py
   self.new_topic_agent = NewTopicAgent()
   workflow.add_node("new_topic_agent", ...)
